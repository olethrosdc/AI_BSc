* Graph basics

** Graphs
- A graph $G = (N, E)$ consists of a set of nodes $N$ and a set of edges $E$.
- An edge $(i,j) \in E$ is a pair of nodes $i, j \in N$. 

** Paths
- A path $x_1, \ldots,  x_t$ is a series of nodes (and edges) in the graph so that
$(x_{i}, x_{i+1}) \in E$.

** Edge Labels
To assign meaning to the graph, we can assign labels to the edges:
- Action (each edge corresponds to an agent action)
- Costs/reward (the cost/gain of each action)
